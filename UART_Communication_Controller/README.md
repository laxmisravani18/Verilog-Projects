# UART Communication Controller (Verilog)

## Overview

This project implements a **UART Communication Controller** using Verilog HDL.
UART is a serial communication protocol used for transmitting and receiving data between devices.

The design includes transmitter, receiver, and baud rate generator modules.


## Features

* UART Transmitter
* UART Receiver
* Baud Rate Generator
* FSM-based control
* Serial communication
* Timing control



## Modules

* `uart_baudrate_generator.v` – Generates baud rate clock
* `uart_transmitter.v` – UART transmitter
* `uart_receiver.v` – UART receiver
* `uart_controller.v` – Top module
* `tb_uart.v` – Testbench
* `uart_waveform.png


## Concepts Used
